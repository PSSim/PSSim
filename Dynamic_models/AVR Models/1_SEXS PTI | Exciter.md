# Excitation System Equations
The following equations describe the excitation system:

$$
\frac{dE_{fd}}{dt} = \frac{1}{T_{e}} \left( K_{a} \left( V_{r} + \frac{T_{a}}{T_{b}} (V_{ref}-V_{t}) \right) -E_{fd} \right)
$$
